# Worktree Context Menu User Stories

## Commit Menu

- As a user inspecting an old commit, I can create a new branch and linked worktree from that commit so I can experiment without moving my current checkout.
- As a user reviewing a commit, I can compare that commit with any existing worktree so I can see what changed in a live working directory.

## Branch Menu

- As a user viewing a branch, I can create a linked worktree from it so I can work on it without switching my current repository.
- As a user viewing a branch that already has a worktree, I can open or reveal that worktree directly from the branch menu.
- As a user viewing a branch, I can compare it with any worktree so I can inspect the branch tip against another working directory.
- As a user viewing a branch that already has a worktree, I can pull, push, lock, unlock, or remove that worktree from the branch menu.
