# Look Graph Stories

High-level sequence: [look-graph-sequence.mermaid](look-graph-sequence.mermaid)

## Story: Load Repository Graph

As a developer, I want Look Graph to show commits, branches, tags, submodules, and worktrees for the selected repository.

Implementation: [load-repository-graph.mermaid](load-repository-graph.mermaid)

## Story: Navigate Into Submodule Graph

As a developer, I want to open a submodule graph as a repository graph so that submodules behave as real repositories.

Implementation: [navigate-submodule-graph.mermaid](navigate-submodule-graph.mermaid)

## Story: Create Worktree From Branch

As a developer, I want to create a linked worktree from a branch so that I can work on multiple checkouts of the same repository.

Implementation: [create-worktree-from-branch.mermaid](create-worktree-from-branch.mermaid)

## Story: Background Auto-Refresh Without Blink

As a developer, I want Look Graph to auto-refresh without visually rebuilding unchanged data.

Implementation: [background-auto-refresh.mermaid](background-auto-refresh.mermaid)
